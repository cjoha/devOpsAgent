# Commercial Telemetry Runbooks

This folder contains runbooks for commercial telemetry providers (e.g., Splunk).

## Runbook Structure

| Runbook | Purpose | Integration |
|:--------|:--------|:------------|
| topology.md | Maps AWS resources to telemetry providers | All |
| metrics-splunk.md | Query metrics from Splunk | Splunk |
| logs-splunk.md | Query logs from Splunk | Splunk |
| activity-jira.md | Find changes via Jira | Jira |
| activity-github.md | Find code changes via GitHub | GitHub |
| activity-confluence.md | Find documentation | Confluence |
| communication-slack.md | Incident communication | Slack |
